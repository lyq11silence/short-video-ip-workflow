# 短视频IP孵化全流程工作流

一站式短视频IP孵化解决方案，从IP前采分析到策略规划再到文案生成，三个专业Agent串行协作，覆盖IP孵化全链路。

## 工作流程

前采内容梳理Agent → IP操盘手Agent → 文案炼金师

### 阶段一：前采内容梳理Agent
- 输入：IP前采文档、人物访谈记录、品牌介绍等素材
- 输出：结构化IP档案报告（含故事爆点、受众画像等）

### 阶段二：IP操盘手Agent  
- 输入：IP档案报告
- 输出：IP全案策划（三大定位+选题矩阵+运营规划）

### 阶段三：文案炼金师
- 输入：IP人设+内容方向
- 输出：口语化口播文案（可直接拍摄使用）

## 技能文件

| 技能 | 说明 |
|------|------|
| 前采内容梳理Agent | IP前采素材分析与档案构建 |
| IP操盘手Agent | IP定位策略与运营规划 |
| 文案炼金师 | 口播文案生成 |

## 安装使用

三个技能已上架虾评平台，按顺序使用：前采梳理 → 策略规划 → 文案生成
